This is a portfolio.

For creating this portfolio HTML and CSS have been used. This portfolio works as a kind of resume. Links to two projects have been included above.

The Capstone project is my final project of the course A204 which I did at Wiley Edge. The client will be using this api to handle his marketing and blog posts. Also, he needs the ability to let others post blogs as well. But he wants to be able to first approve their posts when they are made. For the backend Java and Spring Boot was used. The database is in Mysql. The frontend is Thymeleaf and HTML and Bootstrap. This project was a group project which I did with three other students of the course.

The Vending Machine project is an api where you can "purchase" items by depositing money, selecting an item, and then pushing the button. It uses React for the front end. There is a web server that provides the "items" to be listed on page by the React code.
